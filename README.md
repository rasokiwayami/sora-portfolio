# Sora Portfolio

宮脇蒼空の制作ポートフォリオです。

大学で学びながら、手元で必要になったものや、試してみたいものを小さく作っています。このサイトでは、公開できる制作物と、その制作過程をまとめています。

## このサイト

完成したものだけを並べるよりも、考えて、作って、直した跡が分かる場所にしたいと思って作っています。

今はまだ、大きな実績を並べる段階ではありません。作ったもの、途中で迷ったところ、次に試したいことを少しずつ置いていきます。

## 中にあるもの

- 制作物: GitHubで見られるものや、今見せられる制作
- 制作過程: このサイトを作る中で迷って直したところ
- 活動ログ: 短い制作メモ
- 保管庫: 公開できる範囲に絞った制作の概要

## 開発

```bash
npm install
npm run dev
```

ローカルでは `http://127.0.0.1:5174/` で確認できます。

本番用のビルド:

```bash
npm run build
```
